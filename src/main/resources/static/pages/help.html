 Get started with Neo4j database exploration 


example:
Considering the following relationship let's define the fundamental parts of Neo4j Graph DB:

 (AUTHOR{name})-[WROTE{location}]->(BOOK{title}) 

       - node label: AUTHOR, BOOK
       - relationship type: WROTE
       - node property name: name, title
       - relationship property name: location
